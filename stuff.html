Creation of this site

This website was created using html and css that I typed up myself

I then set up a server on my Raspberry Pi to host the webpage using Apache. I set up the port forwarding on my router so that connections from the outside world would get sent to the Raspberry Pi.

Next I went and purchased a domain name (atypmatrices.info) so people don't have to type in my IP address. I set up the domain name so it will automatically forward to my IP, where my server is hosted
